Tutorial de Comercialização do Sistema

Um guia passo a passo para transformar seu sistema em um produto vendável e seguro.

Parte 1: Seu Servidor de Licenças (Privado)

Esta é a sua central de controle. Crie estes componentes na sua própria conta Google. Eles **nunca** serão compartilhados com os clientes.

Passo 1.1: Criar a Base de Dados de Licenças

Crie uma nova Planilha Google privada chamada `[MESTRE] Licenças Gasto Certo`. Na primeira aba, renomeie para `License_DB` e adicione os seguintes cabeçalhos:

  • `A: LicenseKey`
  • `B: BuyerEmail`
  • `C: Status` (Valores: Disponível, Ativa, Expirada, Bloqueada)
  • `D: ActivationDate`
  • `E: ActivatedUserEmail`
  • `F: ActivationCount`
  • `G: MaxActivations`

Passo 1.2: Criar o Script do Servidor

Crie um novo projeto Apps Script. No arquivo `Code.gs`, cole o código abaixo. Ele funcionará como sua API de validação.

const LICENSE_DB_ID = "ID_DA_SUA_PLANILHA_MESTRE"; 
const SHEET_NAME = "License_DB";

function doGet(e) {
  const params = e.parameter;
  const licenseKey = params.key;
  const userEmail = params.email;

  if (!licenseKey || !userEmail) {
    return createJsonResponse({ success: false, message: "Parâmetros inválidos." });
  }

  const lock = LockService.getScriptLock();
  lock.waitLock(30000);

  try {
    const sheet = SpreadsheetApp.openById(LICENSE_DB_ID).getSheetByName(SHEET_NAME);
    const data = sheet.getDataRange().getValues();
    
    for (let i = 1; i < data.length; i++) {
      if (data[i][0] === licenseKey) {
        const row = i + 1;
        const status = data[i][2];
        const activationCount = parseInt(data[i][5]) || 0;
        const maxActivations = parseInt(data[i][6]) || 1;

        if (status === "Bloqueada" || status === "Expirada") {
          return createJsonResponse({ success: false, message: "Licença bloqueada ou expirada." });
        }

        if (activationCount >= maxActivations && data[i][4] !== userEmail) {
          return createJsonResponse({ success: false, message: "Limite de ativações atingido." });
        }
        
        if (status === "Disponível") {
          sheet.getRange(row, 3).setValue("Ativa");
          sheet.getRange(row, 4).setValue(new Date());
          sheet.getRange(row, 5).setValue(userEmail);
          sheet.getRange(row, 6).setValue(1);
          return createJsonResponse({ success: true, message: "Licença ativada com sucesso!" });
        }
        
        if (status === "Ativa" && data[i][4] === userEmail) {
          return createJsonResponse({ success: true, message: "Licença validada." });
        } else {
          return createJsonResponse({ success: false, message: "Esta licença já está em uso por outra conta Google." });
        }
      }
    }
    return createJsonResponse({ success: false, message: "Chave de licença inválida." });
  } catch (err) {
    return createJsonResponse({ success: false, message: "Erro no servidor de licenciamento." });
  } finally {
    lock.releaseLock();
  }
}

function createJsonResponse(obj) {
  return ContentService.createTextOutput(JSON.stringify(obj))
    .setMimeType(ContentService.MimeType.JSON);
}

Passo 1.3: Implantar a API de Licenciamento

No editor do Script do Servidor, clique em `Implantar` > `Nova implantação` com as seguintes configurações:

  • **Tipo:** `App da Web`
  • **Executar como:** `Eu (seu e-mail)`
  • **Quem pode acessar:** `Qualquer pessoa` (CRUCIAL!)

Clique em `Implantar` e copie a **URL do App da Web**. Guarde-a, você precisará dela na Parte 2.

Parte 2: Preparando o Produto para o Cliente

Agora, vamos modificar o script da sua planilha principal (a que será copiada pelos clientes) para que ela verifique a licença antes de funcionar.

Passo 2.1: Criar o Arquivo de Verificação

No projeto Apps Script da sua planilha de produto, crie um novo arquivo chamado `LicenseCheck.gs` e cole o código abaixo. Lembre-se de inserir a URL da sua API que você copiou na Parte 1.

const LICENSE_SERVER_URL = "URL_DA_SUA_API_DE_LICENCIAMENTO_AQUI";

function activateProduct() {
  const ui = SpreadsheetApp.getUi();
  const response = ui.prompt('Ativação do Produto', 'Por favor, insira sua chave de licença:', ui.ButtonSet.OK_CANCEL);

  if (response.getSelectedButton() == ui.Button.OK) {
    const licenseKey = response.getResponseText().trim();
    if (validateLicense(licenseKey)) {
      ui.alert('Sucesso!', 'Seu produto foi ativado com sucesso. Por favor, recarregue a planilha.', ui.ButtonSet.OK);
    }
  }
}

function validateLicense(licenseKey) {
  const userEmail = Session.getActiveUser().getEmail();
  const url = `${LICENSE_SERVER_URL}?key=${encodeURIComponent(licenseKey)}&email=${encodeURIComponent(userEmail)}`;
  
  try {
    const response = UrlFetchApp.fetch(url, { muteHttpExceptions: true });
    const result = JSON.parse(response.getContentText());

    if (result.success) {
      PropertiesService.getScriptProperties().setProperties({
        'LICENSE_KEY': licenseKey,
        'LICENSE_STATUS': 'VALID',
        'LICENSED_USER': userEmail
      });
      return true;
    } else {
      SpreadsheetApp.getUi().alert('Falha na Ativação', result.message, SpreadsheetApp.getUi().ButtonSet.OK);
      return false;
    }
  } catch (e) {
    SpreadsheetApp.getUi().alert('Erro de Conexão', 'Não foi possível conectar ao servidor de licenciamento.', SpreadsheetApp.getUi().ButtonSet.OK);
    return false;
  }
}

function isLicenseValid() {
  const props = PropertiesService.getScriptProperties();
  const status = props.getProperty('LICENSE_STATUS');
  const user = props.getProperty('LICENSED_USER');
  
  if (status === 'VALID' && user === Session.getActiveUser().getEmail()) {
    return true;
  }
  return false;
}

Passo 2.2: Integrar a Verificação

No arquivo `Code.gs` da sua planilha de produto, substitua a função `onOpen()` e adicione a verificação no início da função `doPost(e)` para proteger o sistema.

// Substitua ou crie a função onOpen()
function onOpen() {
  const ui = SpreadsheetApp.getUi();
  if (isLicenseValid()) {
    ui.createMenu('Gasto Certo')
      .addItem('Configuração Inicial', 'showSetupUI')
      .addToUi();
  } else {
    ui.createMenu('Gasto Certo')
      .addItem('⚠️ Ativar Produto', 'activateProduct')
      .addToUi();
  }
}
// Adicione no início da função doPost(e)
function doPost(e) {
  if (!isLicenseValid()) {
    logToSheet("Acesso bloqueado via doPost: Licença inválida.", "ERROR");
    return; 
  }
  // ... resto do seu código ...
}
